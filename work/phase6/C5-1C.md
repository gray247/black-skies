# C5-1C — Dialogue/Narration Metric (Backend Only)
# This task corresponds to the C5-1C checklist group inside:
#   docs/phase6_analytics_refinement_notes.md

# Objective:
# Implement the Dialogue/Narration ratio metric for the analytics backend.
# This metric must rely ONLY on helpers implemented in C5-1A.
# No UI changes. No endpoint changes. Backend only.

Read first:
  docs/phase6_analytics_refinement_notes.md
  app/backend/services/analytics/text_utils.py
  app/backend/services/analytics/pipeline.py
  app/backend/services/analytics/readability.py (added in C5-1B)
  app/backend/services/analytics/*

Requirements (from C5-1C spec):

1) Dialogue Detection (strict rules)
   - Dialogue = text STRICTLY inside quotes:
         "…"
         '…'
   - Use the quote-span extraction helper created in C5-1A.
   - The function must support multiple dialogue zones.
   - Only well-formed outermost quote pairs count.
   - Malformed or incomplete quote regions are ignored.

2) Narration
   - Narration = all text NOT inside valid quote spans.
   - Narration and dialogue must be disjoint.

3) Ratio Calculation
   - Compute:
         dialogue_char_count
         narration_char_count
   - Convert to token counts IF useful, but final ratio can be char-based.
   - Return:
         dialogue_ratio   = dialogue_chars   / total_chars
         narration_ratio  = narration_chars  / total_chars
   - Guarantee:
         abs((dialogue_ratio + narration_ratio) - 1.0) < 0.001

4) API Contract
   Implement function:
       compute_dialogue_narration(text: str) -> dict
   Returns:
       {
         "dialogue_ratio": float,
         "narration_ratio": float,
         "dialogue_chars": int,
         "narration_chars": int
       }

5) Integration
   - Add this metric to the analytics pipeline:
         app/backend/services/analytics/pipeline.py
   - Include results in both:
         /analytics/summary
         /analytics/scenes
   - DO NOT modify UI or renderer here.

6) Error Handling
   - If text is empty → return 0.0 / 1.0 narration.
   - If no dialogue spans → dialogue = 0.
   - If everything is inside quotes (rare) → narration = 0.
   - Fail gracefully on malformed quote spans.

7) Documentation

         Mark C5-1C as completed.
   - Update work/phase6/phase6_tracker.md:
         Check off C5-1C.
   - Add a brief note in work/phase6/verification_notes.md.

8) Tests (optional but recommended)
   Create or update tests:
      app/backend/tests/test_dialogue_narration.py

When finished, print:

  - "Pipeline updated to include dialogue/narration"
  - "Tracker + refinement notes updated"
