# C5-1F — Insights UI Integration (Renderer)
# This task corresponds to C5-1F in:
#   docs/phase6_analytics_refinement_notes.md

# Objective:
# Integrate all newly implemented backend analytics metrics into the
# Insights panel UI. This includes scene cards, tooltips, pacing strips,
# and empty-data handling.

Read first:
  docs/phase6_analytics_refinement_notes.md
  app/renderer/src/panels/Insights/*
  app/renderer/src/components/Insights/*
  app/renderer/src/utils/*
  app/renderer/src/state/*
  app/renderer/src/hooks/*
  app/backend/services/analytics/*  (for understanding data structures)

Requirements:

1) Add new analytics fields to renderer state
   - Add readability composite fields
   - Add dialogue_ratio and narration_ratio
   - Add pacing_raw, pacing_bucket, scene_length
   - Ensure these types are reflected in any TypeScript interfaces

2) Insights Panel Layout
   - Show readability bucket (Easy/Moderate/Dense)
   - Show dialogue/narration ratios
   - Show pacing bucket (Fast/Neutral/Slow)
   - Include raw values inside tooltips

3) Pacing Strip
   - Add a simple horizontal strip with color-coded pacing:
        Fast    = green (or high-contrast equivalent from policies.md)
        Neutral = yellow/neutral
        Slow    = red/orange (or accessible equivalent)
   - Entire strip must be accessible and meet policies.md contrast rules.

4) Scene Cards
   - Each scene card must display:
        • readability bucket
        • dialogue ratio
        • pacing bucket
   - Add hover tooltips showing:
        • avg sentence length
        • pct long sentences
        • type-token ratio
        • dialogue/narration character counts
        • pacing_raw + scene_length

5) Graphs
   - Update pacing/emotion graph (if present) to include pacing data
   - Add fallback behavior for empty analytics (no freeze, no errors)
   - Ensure large numbers of scenes do NOT cause UI lag

6) Empty or Missing Data Handling
   - If analytics fail to load → show a clear message:
        “Analytics unavailable — try refreshing.”
   - If analytics for a scene are incomplete → show “—”
   - Do NOT cause unhandled exceptions

7) Accessibility + Spacing
   - Follow docs/policies.md contrast requirements
   - Ensure labels use readable font size
   - Apply consistent padding to avoid cramped layout

8) Documentation
   - Update docs/phase6_analytics_refinement_notes.md:
         Mark C5-1F completed.
   - Update work/phase6/phase6_tracker.md:
         Check off C5-1F.
   - Add any renderer-specific notes to work/phase6/verification_notes.md.

9) Tests (optional)
   - Update or add renderer tests:
       app/renderer/tests/Insights/*.test.ts
       app/renderer/tests/AnalyticsDashboard.test.ts

When finished, print:
  - "Insights UI updated with readability/dialogue/pacing"
  - "Scene cards + graphs updated"
  - "Empty-data & accessibility handling added"
  - "Tracker + refinement notes updated"
