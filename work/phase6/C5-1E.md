# C5-1E — Analytics Endpoint Wiring (Backend)
# This task corresponds to C5-1E in:
#   docs/phase6_analytics_refinement_notes.md

# Objective:
# Wire all newly-implemented metrics (C5-1B, C5-1C, C5-1D) into the
# /analytics/summary and /analytics/scenes backend endpoints.

Read first:
  docs/phase6_analytics_refinement_notes.md
  app/backend/services/analytics/pipeline.py
  app/backend/services/analytics/readability.py
  app/backend/services/analytics/dialogue_narration.py
  app/backend/services/analytics/pacing.py
  app/backend/main.py  (or wherever FastAPI routes are registered)
  app/backend/services/utils/*

Requirements:

1) Update analytics pipeline objects
   - Ensure scene-level analytics dict includes:
        readability: {avg_sentence_len, pct_long, ttr, bucket}
        dialogue_ratio
        narration_ratio
        pacing_raw
        pacing_bucket
        scene_length
   - Ensure summary analytics aggregates OR passes through scene-level values
     exactly as intended.

2) Update /analytics/scenes endpoint
   - Return a list of per-scene analytics including all new metrics.
   - Must NOT break existing renderer expectations.
   - Must handle missing or empty scenes gracefully.

3) Update /analytics/summary endpoint
   - Include high-level aggregated metrics when appropriate.
   - Must include:
        - readability composite for the whole project (approx. merged text)
        - global dialogue/narration ratios
        - pacing distribution summary
   - If aggregation logic not present, implement minimal aggregation:
        * concatenate all scene texts
        * recompute using helpers
        * OR compute weighted averages, whichever is simpler/safer.

4) Router integration
   - Ensure the router file (FastAPI) exposes:
        GET /analytics/scenes
        GET /analytics/summary
   - Ensure both use updated pipeline functions.

5) Error handling
   - Missing project directory → return empty analytics.
   - Corrupt scene content → skip that scene, log warning.
   - Never crash the backend.

6) Documentation
   - Update docs/phase6_analytics_refinement_notes.md:
         Mark C5-1E completed.
   - Update work/phase6/phase6_tracker.md:
         Check off C5-1E.
   - Add brief mention in work/phase6/verification_notes.md.

7) Tests (optional but recommended)
   Add or update:
       app/backend/tests/test_analytics_endpoints.py

When finished, print:
  - "Analytics endpoints updated"
  - "Scenes + summary now include readability, dialogue/narration, pacing"
  - "Tracker + refinement notes updated"
